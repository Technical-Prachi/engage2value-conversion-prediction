# Engage2Value: Clicks to Conversions (ML Project)

## Overview
This project focuses on predicting user conversions based on clickstream and engagement data using machine learning techniques.

## Problem Statement
To identify whether a user will convert based on interaction data and improve marketing effectiveness.

## Tech Stack
- Python
- pandas, NumPy
- scikit-learn

## Approach
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Model building (classification)
- Model evaluation

## Result
Achieved a best model score of 0.59 on the validation dataset.

## Learnings
- Understanding user behavior through data
- Feature importance in conversion prediction
- Improving model performance through tuning
